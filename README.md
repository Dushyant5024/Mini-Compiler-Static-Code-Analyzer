# Mini Compiler & Static Code Analyzer

A modern compiler simulation and static analysis platform that visualizes the complete compilation pipeline including lexical analysis, parsing, semantic checking, optimization, and intermediate code generation.

---

# Features

- ⚡ Lexical Analysis & Token Generation
- 🌳 Abstract Syntax Tree (AST) Explorer
- 🧠 Semantic Analysis & Symbol Table
- 🔍 Static Code Analysis
- 🚀 Code Optimization Passes
- ⚙️ Intermediate Representation (IR) Code Generation
- 📊 Compilation Metrics & Performance Reports
- 🎨 Interactive Dark-Themed Dashboard
- 📁 Full Compilation Report Export

# Pipeline Workflow

```txt
Source Code
     ↓
Lexer
     ↓
Parser
     ↓
AST Generator
     ↓
Semantic Analyzer
     ↓
Static Analyzer
     ↓
Optimizer
     ↓
IR Generator
     ↓
Final Report
```
